List Tags
Tag Name Description Example link
<ul> Unordered lists The wrapper element of unordered-lists /
<ol> Ordered lists The wrapper element of ordered-lists /
<li> List element The list element /
<dl> Description list The wrapper element of description-lists /
<dt> Data term The defining element within a description-lists /
<dd> Term name The describing element within a defining element of a description-lists /
Table Tags
Tag Name Description Example link
<table> Table The wrapper element of tables /
<caption> Caption The name of the table /
<thead> Table head The set of rows defining the column headers in a table /
<tbody> Table body The set of rows containing the actual table data /
<tr> Table row The table row wrapper /
<th> Table head data The table head row container /
<td> Table data The table row container /
<tfoot> Table footer The table footer wrapper /

Attribute Name Description Example link

Tag Name Description Example link
<form> Form The wrapper element of forms /

<input> Input Tag describing what is going to go in the form and how it interacts with the code /
Forms Attributes
Attribute Name Description Example link
for For The type of the form. "username", "password", "html", "javascript"...
type Type The type of the input "text", "password", "submit", "radio", "checkbox", "number", "file", "date"...
action Action The form needs an action from the user /
name Name Name of the input once in memory /
value Value Value type of the input once in memory "HTML", "javascript"...
id ID Name of the input used within the html code /
